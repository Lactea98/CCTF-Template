# Casper CTF Template



### 1. 개요

------------


이 템플릿은 [CASPER](https://casper.or.kr) 에서 사용할 CTF 템플릿 입니다. 

개발 목적은 CASPER 만의 독자적인 CTF 템플릿이 있으면 좋겠다는 생각으로 개발을 시작했습니다.
